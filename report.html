Implementação: Detecção de Discrepantes¶

Identificar dados discrepantes extremamente importante no passo de pr-processamento de dados de qualquer anlise. A presena de discrepantes podem enviesar resultados que levam em considerao os pontos de dados. H muitas "regras bsicas" que constituem um discrepante em um conjunto de dados. Aqui usaremos o Mtodo Turco para identificar discrepantes: Um passo do discrepante calculado 1,5 vezes a variao interquartil (IQR). Um ponto de dados com um atributo que est alm de um passo de um discrepante do IQR para aquele atributo, ele considerado anormal.

No bloco de cdigo abaixo, voc vai precisar implementar o seguinte:

  • Atribuir o valor do 25 percentil do atributo dado para o Q1. Utilizar np.percentile para isso.
  • Atribuir o valor do 75 percentil do atributo dado para o Q3. Novamente, utilizar np.percentile.
  • Atribuir o clculo de um passo do discrepante do atributo dado para o step.
  • Remover opcionalmentos os pontos de dados do conjunto de dados ao adicionar ndices lista de outliers.

NOTA: Se voc escolheu remover qualquer discrepante, tenha certeza que a amostra de dados no contm nenhum desses pontos!
Uma vez que voc executou essa implementao, o conjunto de dado ser armazenado na varivel good_data!Once you have performed this implementation, the dataset will be stored in the variable .

outliers = []

    # TODO: Calcule Q1 (25 percentil dos dados) para o atributo dado
    Q1 = np.percentile(log_data[feature], 25)
    
    # TODO: Calcule Q3 (75 percentil dos dados) para o atributo dado
    Q3 = np.percentile(log_data[feature], 75)
